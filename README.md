# ORSystem - Campus Event Budget Allocation System

## Project Description

ORSystem is a web-based Linear Programming Decision Support System developed using PHP and MySQL.

The system helps student clubs optimize budget allocation for activities such as catering, marketing, and equipment while maximizing the effectiveness of campus events.

## Features

* User Registration
* User Login
* Dashboard
* Linear Programming Input Form
* Optimization Solver
* Result Display
* History Management
* MySQL Database Storage

## Technologies Used

* PHP
* MySQL
* Bootstrap 5
* HTML
* CSS

## How to Run

1. Install XAMPP
2. Start Apache and MySQL
3. Create database: `orsystem_db`
4. Import `orsystem_db.sql`
5. Place project folder in `htdocs`
6. Open:

http://localhost/ORSystem/
